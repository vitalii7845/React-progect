# Google Calendar (React)

## [Site link](https://vitalii7845.github.io/React-progect/)

## The tech stack is:

- React
- React Hooks
- JS (ES6)
- Sass (Scss)
- HTML5
- Webpack
- REST API

## Commands:

- `npm start` - to start project
- `npm run build` - to build production
- `npm run lint` - check syntacks with eslint
- `npm run test` - test with jest
